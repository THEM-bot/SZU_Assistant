# SZU_Assistant
深圳大学内部网油猴脚本，实现各种方便的功能
SZU Assistant.user.js为脚本文件,Tampermonkey为油猴，如果你不会安装，可以拿上这两个文件去狠狠拷打AI，安装成功后访问深大内部网几秒后浏览器标签名前面会出现“[SZU]”字样


# 目前主要功能包括：
1.几个常用网站快捷入口

2.宿舍用电快速查询

3.绑定账号信息后自动登录以下网站：统一身份认证，办事大厅，校园网续费网站，本科选课网站（这个网站因未知原因自动输入账号密码很慢），webvpn

4.办事大厅中自动全五星评教

5.去除了公文通的隐藏水印（会显示学生姓名和学号）

6.公文通列表只看各个学院的公文选项
